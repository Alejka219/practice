Аннотация: Проект направлен на создание веб-сервиса для планирования задач с автоматической расстановкой приоритетов на основе машинного обучения. Целевая аудитория – студенты и офисные работники.

О проекте

Планировщик позволяет создавать задачи, устанавливать дедлайны, получать рекомендации по оптимизации расписания. Используемые технологии: Python (Flask), HTML/CSS, JavaScript, SQLite. Проект реализуется в рамках дисциплины «Проектная деятельность».

Основные функции: регистрация пользователей, CRUD задач, уведомления, аналитика времени.
